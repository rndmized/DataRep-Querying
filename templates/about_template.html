The Races and Classes used in this Character Creation App are the ones that you can obtain for free in the Dungeons and dargons website. For more diverse Races and classes we encourage you to buy the Player Character Handbook!

The DnD 5e Character Creation Project

This webapp has been created for Data Representation's and Querying Module for 3rd Year Software Development at Galway-Mayo Technical Institute in Ireland. The main goal was to create a single web page application using Python, Flask, Bootstrap, and in this case JQuery. It also uses MongoDB as a database!
